# Datenerhebungs-Checkliste – ADR 0002

## Anleitung

1. Browser-Konsole öffnen (F12 → Console)
2. Karte einprägen
3. Stapel wählen gemäß Szenario
4. Console-Ausgabe `[ADR-0002]` kopieren/notieren
5. Auf Karte klicken → neues Spiel startet automatisch
