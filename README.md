# The magician

Dieses kleine Elm Projekt zeigt dir einen kleinen zauberer, der tatsächlich deune Gedanken lesen kann. Er teilt vor deinen Augen Karten auf drei Stapel aus und du merkst dir eine Karte und in welchem Stapel sie gelandet ist. Durch klick auf den Stapel mit der Karte wiederhokt der Zsuberer das drei Mal. Danach kann er dir due Karte präsentierrn die du dir gemerkt hast...
